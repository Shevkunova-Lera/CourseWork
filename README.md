# CourseEork
A Python desktop application for optimal production planning and cost minimization using linear programming

## Features
* Cost Minimization: Calculates the most cost-effective distribution of normal, overtime, and reserve manufacturing modes.
* Linear Programming: Uses `scipy.optimize.linprog` to solve the transportation matrix.
* Interactive GUI: Built with `PyQt5` for easy input of quarterly demand, production limits, and cost parameters.

## Technologies Used
* Python
* SciPy & NumPy
* PyQt5
